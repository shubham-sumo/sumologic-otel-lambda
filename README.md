# Sumo Logic Distribution for OpenTelemetry Lambda Layers

`sumologic-otel-lambda` publishes preconfigured [OpenTelemetry Lambda](https://github.com/open-telemetry/opentelemetry-lambda) layers which provide instrumentation for AWS Lambda functions.
Released `sumologic-otel-lambda` layers are available:

- Java wrapper layer contains OpenTelemetry Java `v2.27.0` and OpenTelemetry Collector `v0.151.0`. Please see list of [lambda layers](https://github.com/SumoLogic/sumologic-otel-lambda/blob/release-java-v2.20.0/java/README.md).

- NodeJS layer contains OpenTelemetry JavaScript SDK `v2.7.0` and OpenTelemetry Collector `v0.151.0`. Please see list of [lambda layers](https://github.com/SumoLogic/sumologic-otel-lambda/blob/release-nodejs-v2.1.0/nodejs/README.md).

- Python layer contains OpenTelemetry Python SDK `v1.40.0` with instrumentation `v0.61b0` and OpenTelemetry Collector `v0.148.0`. Please see list of [lambda layers](https://github.com/SumoLogic/sumologic-otel-lambda/blob/release-python-v1.40.0/python/README.md).
